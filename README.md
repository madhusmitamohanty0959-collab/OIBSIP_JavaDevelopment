# Task 1 - Online Reservation System

## Objective
To develop a console-based Online Reservation System using Java.

## Features
- User Login
- Reservation Form
- Ticket Booking
- Ticket Cancellation
- Display Reservation Details

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac OnlineReservationSystem.java

2. Run the program:
   java OnlineReservationSystem
