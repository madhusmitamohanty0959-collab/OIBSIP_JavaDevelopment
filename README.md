# Task 1 - Online Reservation System

## Objective
To develop a console-based Online Reservation System using Java.

## Features
- User Login
- Reservation Form
- Ticket Booking
- Ticket Cancellation
- Display Reservation Details

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac OnlineReservationSystem.java

2. Run the program:
   java OnlineReservationSystem
   
# Task 2 - Number Guessing Game

## Objective
To develop a Number Guessing Game using Java where the user has to guess a randomly generated number within a limited number of attempts.

## Features
- Random number generation
- User input through console
- Hints for higher or lower guesses
- Limited attempts
- Score calculation
- User-friendly interface

## Technologies Used
- Java
- VS Code

## How to Run

1. Compile the program:
   javac NumberGuessingGame.java

2. Run the program:
   java NumberGuessingGame

## Output
The program generates a random number and asks the user to guess it. Based on the input, it provides hints and displays the result after the correct guess or when attempts are exhausted.

## Learning Outcome
This project helped in understanding Java concepts such as loops, conditional statements, random number generation, user input handling, and program logic.
